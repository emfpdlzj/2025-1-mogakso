# 2025 하계 모각소 공부 ✨
> **주제:** C 언어 · 리눅스 커널 기초 · TinyML 기초  
> **최종 목표:** 임베디드 환경에서 **TinyML** 간단 구현

---

## 📌 개요
방학 동안 아주대학교 모각소(모여서 각자 공부)로 진행한 학습 내용을 정리한 저장소입니다.  
C 언어와 리눅스 기본기를 다지고, TinyML 구현을 위한 머신러닝 핵심 개념을 학습했습니다.

---

## ✅ 이번 학습에서 얻은 것
- C 언어 문법·포인터·메모리 모델 **재정비**
- 리눅스 환경에서의 개발 워크플로우 **기초 내공**
- TinyML 구현을 위한 **지도학습/최적화 핵심 개념** 정리
- 초소형 모델의 **수집→학습→배포 파이프라인** 개관

---

## 🧭 학습 로드맵
1. **C 언어 기초 재정비** → 문법/메모리/포인터 감각 회복  
2. **리눅스 기본기** → 커맨드/프로세스/파일시스템 기초  
3. **ML 핵심 개념** → 회귀·분류 등 TinyML 구현에 필요한 최소 이론  
4. **TinyML 개론** → 초소형 모델, 배포 흐름 이해
